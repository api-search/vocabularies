# vocabularies.apis.io — Provider Vocabularies

[**vocabularies.apis.io**](https://vocabularies.apis.io) indexes the structured vocabulary documents associated with each provider — resources, actions, personas, and domains that describe an API\'s conceptual surface in shared terminology.

**Live site:** [https://vocabularies.apis.io](https://vocabularies.apis.io)

## What's in this repo

- **`vocabulary.json (root)`** — a single aggregated `vocabulary.json` file containing every provider vocabulary in the network, plus per-provider browse pages rendered from it.
- **`index.html`, `_layouts/`, `_includes/`, `assets/`** — the browse and detail pages
- **`_config.yml`** — Jekyll site configuration

## How it's built

This collection is regenerated by [`network/scripts/build.py`](https://github.com/api-search/network/blob/main/scripts/build.py), which reads the upstream [api-evangelist](https://github.com/api-evangelist) provider repos and emits the records here. The site itself is a Jekyll build deployed via GitHub Pages.
