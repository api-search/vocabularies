# API Search - Vocabularies

This repository stores API vocabulary definitions for the [APIs.io](https://apis.io) search network.

## About

Vocabularies provide standardized terminology and semantic definitions used across APIs indexed by the network, supporting consistent naming and discoverability.

## Related Repositories

- [apis](https://github.com/api-search/apis) - Individual API records
- [json-ld](https://github.com/api-search/json-ld) - JSON-LD semantic vocabularies
- [schemas](https://github.com/api-search/schemas) - JSON Schema references
- [network](https://github.com/api-search/network) - The APIs.io network site
